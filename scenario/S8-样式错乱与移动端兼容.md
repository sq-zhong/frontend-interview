# S8 · 样式错乱与移动端兼容

> 场景排错专项 · 现象 → 排查路径 → 常见根因 → 解决 → 举一反三

## 现象

某元素被遮挡 / 点不到 / 溢出;弹窗层级不对被盖住;PC 正常但移动端错位;不同机型表现不一致;1px 边框忽粗忽细。

## 排查路径

```
① Elements 面板选中出问题的元素:
   - Computed 看最终生效的样式,哪条被覆盖了(划掉的)
   - 看盒模型图:margin/padding/border 是否异常、有没有溢出
② 层级问题(被遮挡/点不到):
   - 查 z-index 和 position;查是否有元素盖在上面(改透明度看)
   - 是不是父级 overflow:hidden 裁剪了
③ 移动端:用真机 / 远程调试(不能只信 PC 模拟器)
   - 看 viewport 设置、DPR、安全区域
④ 优先级问题:样式没生效?查选择器权重(见 C6)、!important
```

## 常见根因(分类)

**1. 层叠与遮挡(见 C4)**
- z-index 不生效——元素没设 position,或困在父级的层叠上下文里(父级 transform/opacity 创建了新层叠上下文);
- 元素被盖住导致"点不到"(上面有透明元素挡着)。

**2. 溢出与裁剪**
- 内容溢出容器(没设 `overflow`、`word-break`、`text-overflow`);
- 父级 `overflow:hidden` 把子级(如下拉菜单、气泡)裁掉了。

**3. 样式优先级(见 C6)**
- 被更高权重的选择器覆盖;组件库样式冲突;`!important` 打架。

**4. 移动端专属**
- viewport 没设 `width=device-width`,页面被缩放;
- **1px 边框**在高 DPR 屏变粗(见 C7);
- 点击 300ms 延迟、点击穿透;
- `100vh` 在移动端含地址栏导致高度跳动(用 `dvh`);
- iOS 安全区域(刘海/底部)遮挡,需要 `env(safe-area-inset-*)`;
- 字体、滚动惯性、fixed 定位在软键盘弹出时的异常。

**5. 盒模型**
- 没统一 `box-sizing: border-box`,加了 padding 后宽度超预期(见 C1)。

## 解决方案速查

| 问题 | 解决 |
|------|------|
| z-index 无效 | 设 position;检查父级层叠上下文(transform/opacity) |
| 下拉被裁剪 | 父级别用 overflow:hidden;或用 Portal 渲染到 body |
| 文本溢出 | `overflow:hidden; text-overflow:ellipsis; white-space:nowrap` |
| 样式被覆盖 | 提升选择器权重 / 调整顺序 / 避免滥用 !important |
| 移动端缩放 | `<meta viewport width=device-width, initial-scale=1>` |
| 1px 边框 | 伪元素 + `transform: scaleY(0.5)`(见 C7) |
| 100vh 跳动 | 用 `dvh` 或 JS 计算真实视口高度 |
| 安全区域 | `padding: env(safe-area-inset-bottom)` |
| 盒模型 | 全局 `* { box-sizing: border-box }` |

## 举一反三(面试如何答)

**Q:一个元素 z-index 设了很大还是被盖住,为什么?**
> 两个常见原因:**① 元素没有设 position**(static 下 z-index 无效);**② 它被困在某个父级的层叠上下文里**——如果父级用了 transform、opacity<1、filter 等,会创建新的层叠上下文,子元素的 z-index 再大也只在父级内部比较,无法超过父级同层的其他元素。解决是把该元素提到更高层级的上下文,或用 Portal 渲染到 body 下。

**Q:移动端适配踩过哪些坑?**
> 常见的有:viewport 没配好导致整体缩放;**1px 边框在 Retina 屏变粗**,要用伪元素 scaleY(0.5);100vh 因地址栏伸缩跳动,改用 dvh;iOS 刘海和底部安全区域要用 env() 适配;还有点击延迟、点击穿透、软键盘顶起 fixed 元素等。**移动端一定要真机调试,不能只信 PC 模拟器。**

## 一句话总结

**样式排错用 Elements+Computed 看谁覆盖了谁、盒模型是否溢出;z-index 无效查 position 和父级层叠上下文(transform/opacity 会新建);下拉被裁用 Portal;移动端坑:viewport、1px 边框(scaleY 0.5)、100vh 跳动(dvh)、安全区域 env()、真机调试;统一 box-sizing:border-box。**

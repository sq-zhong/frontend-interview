# A3 · 双指针与滑动窗口

> 热度:🔥🔥🔥

## 一句话考点

**双指针**用两个指针遍历(对撞/快慢/同向)把 O(n²) 优化到 O(n);**滑动窗口**是同向双指针的典型应用,用于求连续子数组/子串的最优解。

## 原理精讲

### 1. 双指针三种形式

| 形式 | 说明 | 典型题 |
|------|------|--------|
| **对撞指针** | 一头一尾向中间靠拢 | 两数之和(有序)、反转、盛水容器 |
| **快慢指针** | 一快一慢,速度不同 | 链表判环、找中点、删除倒数第 N 个 |
| **同向指针** | 都从左往右,维护区间 | 滑动窗口、原地去重 |

### 2. 滑动窗口

用 left、right 维护一个窗口 `[left, right]`:
- **扩张**:right 右移,把新元素纳入窗口;
- **收缩**:当窗口不满足条件时,left 右移缩小;
- 每个元素最多进出窗口一次,O(n)。

适用:求满足条件的**最长/最短连续子数组/子串**。

## 代码示例

### 对撞指针:两数之和(有序数组)

```js
function twoSum(nums, target) {
  let left = 0, right = nums.length - 1;
  while (left < right) {
    const sum = nums[left] + nums[right];
    if (sum === target) return [left, right];
    else if (sum < target) left++;      // 和太小,左指针右移
    else right--;                        // 和太大,右指针左移
  }
  return [];
}
```

### 快慢指针:链表找中点

```js
function findMiddle(head) {
  let slow = head, fast = head;
  while (fast && fast.next) {
    slow = slow.next;                    // 慢指针走一步
    fast = fast.next.next;               // 快指针走两步
  }
  return slow;                            // fast 到尾,slow 在中点
}
```

### 滑动窗口:无重复字符的最长子串(经典)

```js
function lengthOfLongestSubstring(s) {
  const set = new Set();
  let left = 0, max = 0;
  for (let right = 0; right < s.length; right++) {
    // 窗口内有重复,收缩左边界直到无重复
    while (set.has(s[right])) {
      set.delete(s[left]);
      left++;
    }
    set.add(s[right]);
    max = Math.max(max, right - left + 1);
  }
  return max;
}
```

### 滑动窗口:最小覆盖子串(变长窗口模板)

```js
function minWindow(s, t) {
  const need = new Map();
  for (const c of t) need.set(c, (need.get(c) || 0) + 1);
  let left = 0, valid = 0, start = 0, len = Infinity;
  const window = new Map();
  for (let right = 0; right < s.length; right++) {
    const c = s[right];
    if (need.has(c)) {
      window.set(c, (window.get(c) || 0) + 1);
      if (window.get(c) === need.get(c)) valid++;
    }
    while (valid === need.size) {          // 满足条件,收缩找最小
      if (right - left + 1 < len) { start = left; len = right - left + 1; }
      const d = s[left++];
      if (need.has(d)) {
        if (window.get(d) === need.get(d)) valid--;
        window.set(d, window.get(d) - 1);
      }
    }
  }
  return len === Infinity ? '' : s.substr(start, len);
}
```

## 高频追问

**Q:双指针有哪几种?各自场景?**
A:① 对撞指针(一头一尾向中间,用于有序数组两数之和、反转、盛水容器);② 快慢指针(速度不同,用于链表判环、找中点);③ 同向指针(维护区间,用于滑动窗口、原地去重)。

**Q:滑动窗口的适用场景和思路?**
A:求连续子数组/子串的最长/最短满足某条件的解。思路:right 扩张纳入元素,当窗口满足/不满足条件时 left 收缩,每个元素最多进出一次,O(n)。

**Q:快慢指针如何判断链表有环?**
A:快指针每次两步、慢指针一步。若有环,快指针会在环内追上慢指针(相遇);若无环,快指针先到达 null。相遇后从头和相遇点各一指针同速走可找环入口。

**Q:滑动窗口和暴力解的复杂度差异?**
A:暴力枚举所有子串 O(n²) 甚至 O(n³);滑动窗口每个元素最多被 left 和 right 各访问一次,O(n)。空间通常 O(k)(窗口内字符集)。

**Q:什么时候窗口该收缩?**
A:定长窗口:right 到达窗口大小后每步固定移动 left;变长窗口:当窗口满足条件(求最短)或不满足条件(求最长)时收缩 left。

## 一句话背诵版

**双指针把 O(n²) 降到 O(n):对撞(有序两数之和)、快慢(链表判环/中点)、同向(滑动窗口);滑动窗口 right 扩张、条件满足/不满足时 left 收缩,每元素进出一次 O(n),求连续子串最长/最短;经典题:无重复最长子串、最小覆盖子串。**

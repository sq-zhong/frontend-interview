# A5 · 二叉树遍历

> 热度:🔥🔥🔥

## 一句话考点

二叉树遍历分**深度优先**(前序根左右、中序左根右、后序左右根)和**广度优先**(层序);递归简洁,迭代用栈(DFS)/队列(BFS);**二叉搜索树的中序遍历是升序**。

## 原理精讲

### 1. 四种遍历

| 遍历 | 顺序 | 用途 |
|------|------|------|
| **前序** | 根 → 左 → 右 | 复制树、序列化 |
| **中序** | 左 → 根 → 右 | BST 得升序序列 |
| **后序** | 左 → 右 → 根 | 删除树、计算目录大小 |
| **层序** | 逐层从左到右 | 按层处理、最短路径 |

前中后序的"前中后"指**根节点被访问的时机**。

### 2. 递归 vs 迭代

- **递归**:天然贴合树结构,简洁,但深树可能栈溢出;
- **迭代**:DFS 用栈模拟递归,BFS 用队列。

## 代码示例

### 递归遍历(三种只差一行)

```js
function preorder(root, res = []) {
  if (!root) return res;
  res.push(root.val);           // 根(前序:最先)
  preorder(root.left, res);
  preorder(root.right, res);
  return res;
}
function inorder(root, res = []) {
  if (!root) return res;
  inorder(root.left, res);
  res.push(root.val);           // 根(中序:中间)→ BST 得升序
  inorder(root.right, res);
  return res;
}
function postorder(root, res = []) {
  if (!root) return res;
  postorder(root.left, res);
  postorder(root.right, res);
  res.push(root.val);           // 根(后序:最后)
  return res;
}
```

### 前序迭代(栈)

```js
function preorderIterative(root) {
  if (!root) return [];
  const res = [], stack = [root];
  while (stack.length) {
    const node = stack.pop();
    res.push(node.val);
    // 先压右后压左,保证左先出栈
    if (node.right) stack.push(node.right);
    if (node.left) stack.push(node.left);
  }
  return res;
}
```

### 层序遍历(BFS,队列)

```js
function levelOrder(root) {
  if (!root) return [];
  const res = [], queue = [root];
  while (queue.length) {
    const level = [];
    const size = queue.length;         // 固定当前层节点数
    for (let i = 0; i < size; i++) {
      const node = queue.shift();
      level.push(node.val);
      if (node.left) queue.push(node.left);
      if (node.right) queue.push(node.right);
    }
    res.push(level);                    // 每层一个数组
  }
  return res;
}
```

### 常见衍生题

```js
// 最大深度
function maxDepth(root) {
  if (!root) return 0;
  return 1 + Math.max(maxDepth(root.left), maxDepth(root.right));
}

// 是否对称
function isSymmetric(root) {
  const check = (l, r) => {
    if (!l && !r) return true;
    if (!l || !r || l.val !== r.val) return false;
    return check(l.left, r.right) && check(l.right, r.left);
  };
  return !root || check(root.left, root.right);
}
```

## 高频追问

**Q:前中后序遍历的区别?**
A:指根节点被访问的时机。前序根左右、中序左根右、后序左右根。左右子树的相对顺序不变,只是根的位置不同。

**Q:二叉搜索树用哪种遍历得到有序序列?**
A:中序遍历(左→根→右)。因为 BST 左子树都小于根、右子树都大于根,中序访问顺序恰好是升序。可用此性质验证 BST 或找第 K 小。

**Q:层序遍历怎么按层分组?**
A:BFS 时在每轮循环开始记录当前队列长度(即该层节点数),只处理这么多个节点为一层,把它们的子节点入队作为下一层。

**Q:递归遍历的缺点?如何避免栈溢出?**
A:递归深度等于树高,极端不平衡的树(退化成链)可能栈溢出。可改用迭代(显式栈/队列)或 Morris 遍历(O(1) 空间)。

**Q:DFS 和 BFS 遍历树分别用什么数据结构?**
A:DFS 用栈(递归本质也是调用栈);BFS 用队列(先进先出保证逐层处理)。

## 一句话背诵版

**DFS:前序(根左右)、中序(左根右,BST 得升序)、后序(左右根),递归简洁、迭代用栈;BFS 层序用队列(每轮固定当前层节点数分层);衍生:最大深度、对称、路径等多用递归。**

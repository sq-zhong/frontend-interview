# A7 · 动态规划

> 热度:🔥🔥🔥

## 一句话考点

动态规划(DP)把问题拆成**重叠子问题**,用**状态转移方程**从小到大递推、缓存中间结果避免重复计算;关键三步:**定义状态 → 找转移方程 → 确定初始值和遍历顺序**。

## 原理精讲

### 1. DP 适用条件

- **最优子结构**:大问题的最优解可由子问题最优解推出;
- **重叠子问题**:子问题被重复计算(否则用分治即可)。

### 2. 解题四步

1. **定义状态**:`dp[i]` 表示什么(最难、最关键);
2. **状态转移方程**:`dp[i]` 如何由前面的状态推出;
3. **初始化**:边界值(dp[0]、dp[1]);
4. **遍历顺序**:保证计算 dp[i] 时依赖的状态已算好。

### 3. 优化

- **空间优化**:若 dp[i] 只依赖前几项,可用滚动变量把 O(n) 降到 O(1);
- **记忆化递归**:自顶向下 + 缓存,等价于 DP。

## 代码示例

### 爬楼梯(入门,斐波那契型)

```js
// dp[i] = 到第 i 阶的方法数 = dp[i-1] + dp[i-2]
function climbStairs(n) {
  if (n <= 2) return n;
  let a = 1, b = 2;                    // 滚动变量,O(1) 空间
  for (let i = 3; i <= n; i++) {
    [a, b] = [b, a + b];
  }
  return b;
}
```

### 最大子数组和(Kadane)

```js
// dp[i] = 以 i 结尾的最大连续和 = max(nums[i], dp[i-1]+nums[i])
function maxSubArray(nums) {
  let cur = nums[0], max = nums[0];
  for (let i = 1; i < nums.length; i++) {
    cur = Math.max(nums[i], cur + nums[i]);   // 要么接上前面,要么重新开始
    max = Math.max(max, cur);
  }
  return max;
}
```

### 0-1 背包(经典二维)

```js
// dp[i][w] = 前 i 个物品、容量 w 时的最大价值
function knapsack(weights, values, W) {
  const n = weights.length;
  const dp = Array.from({ length: n + 1 }, () => new Array(W + 1).fill(0));
  for (let i = 1; i <= n; i++) {
    for (let w = 0; w <= W; w++) {
      dp[i][w] = dp[i - 1][w];          // 不选第 i 个
      if (w >= weights[i - 1]) {
        dp[i][w] = Math.max(dp[i][w],
          dp[i - 1][w - weights[i - 1]] + values[i - 1]);  // 选第 i 个
      }
    }
  }
  return dp[n][W];
}
```

### 最长递增子序列(LIS)

```js
// dp[i] = 以 nums[i] 结尾的 LIS 长度
function lengthOfLIS(nums) {
  const dp = new Array(nums.length).fill(1);
  let max = 1;
  for (let i = 1; i < nums.length; i++) {
    for (let j = 0; j < i; j++) {
      if (nums[j] < nums[i]) dp[i] = Math.max(dp[i], dp[j] + 1);
    }
    max = Math.max(max, dp[i]);
  }
  return max;                            // O(n²);可用二分优化到 O(n log n)
}
```

## 高频追问

**Q:动态规划的解题步骤?**
A:① 定义状态 dp[i] 的含义;② 推导状态转移方程(dp[i] 如何由前面状态得到);③ 确定初始值/边界;④ 确定遍历顺序保证依赖已算好。最后可做空间优化。

**Q:DP 和递归/分治的区别?**
A:分治的子问题相互独立;DP 的子问题**重叠**(会重复计算),所以需要缓存(自底向上递推或记忆化递归)。DP = 分治 + 重叠子问题 + 最优子结构。

**Q:什么是最优子结构和重叠子问题?**
A:最优子结构指大问题最优解由子问题最优解组成;重叠子问题指求解过程中相同子问题被多次计算。两者都满足才适合用 DP。

**Q:如何优化 DP 的空间?**
A:若 dp[i] 只依赖前有限个状态(如 dp[i-1]、dp[i-2]),可用几个滚动变量代替整个数组,把空间从 O(n) 降到 O(1)(如爬楼梯、最大子数组和)。

**Q:记忆化递归和 DP 关系?**
A:记忆化递归是自顶向下 + 缓存已解子问题;DP 数组是自底向上递推。两者时间复杂度相同,解决的是同类问题,可互相转换。

## 一句话背诵版

**DP 解决有最优子结构 + 重叠子问题的问题;四步:定义状态 dp[i]→写转移方程→定初始值→定遍历顺序;dp[i] 只依赖前几项时用滚动变量把空间降到 O(1);经典:爬楼梯、最大子数组和(Kadane)、0-1 背包、LIS。**

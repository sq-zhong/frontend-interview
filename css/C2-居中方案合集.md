# C2 · 居中方案合集

> 热度:🔥🔥

## 一句话考点

水平垂直居中是布局基本功;首选 **Flex**(`justify-content + align-items: center`),现代还可用 **Grid**(`place-items: center`),兼容老场景用 **absolute + transform**。

## 原理精讲

居中要区分:水平居中、垂直居中、水平垂直居中;元素是行内/块级/定宽/不定宽,方案不同。

## 代码示例

### 一、水平居中

```css
/* 行内元素 */
.parent { text-align: center; }

/* 块级定宽元素 */
.child { width: 200px; margin: 0 auto; }

/* Flex(不定宽通用) */
.parent { display: flex; justify-content: center; }
```

### 二、垂直居中

```css
/* 单行文本:行高等于高度 */
.box { height: 50px; line-height: 50px; }

/* Flex */
.parent { display: flex; align-items: center; }
```

### 三、水平垂直居中(重点,5 种)

```css
/* 方案1:Flex —— 最推荐 */
.parent {
  display: flex;
  justify-content: center;
  align-items: center;
}

/* 方案2:Grid —— 最简洁 */
.parent {
  display: grid;
  place-items: center;
}

/* 方案3:absolute + transform —— 不需知道宽高,兼容性好 */
.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

/* 方案4:absolute + margin auto —— 定宽高 */
.child {
  position: absolute;
  top: 0; right: 0; bottom: 0; left: 0;
  width: 200px; height: 100px;
  margin: auto;
}

/* 方案5:absolute + 负 margin —— 需已知宽高 */
.child {
  position: absolute;
  top: 50%; left: 50%;
  width: 200px; height: 100px;
  margin-top: -50px; margin-left: -100px;
}
```

## 高频追问

**Q:居中首选什么方案?为什么?**
A:Flex。代码少、不需知道子元素宽高、响应式友好,是现代布局标准做法。单个元素居中 Grid 的 `place-items: center` 更简洁。

**Q:`transform: translate(-50%, -50%)` 里的百分比是相对谁?**
A:相对**元素自身**的宽高。而 `top/left: 50%` 是相对**父元素**。两者配合把元素中心对齐到父元素中心,因此不需要知道子元素具体尺寸。

**Q:absolute + margin auto 为什么能居中?**
A:四个方向都设 0 会拉伸元素,但设了固定宽高后有剩余空间,margin auto 会在两个方向平分剩余空间,实现居中。

**Q:不定宽高元素垂直居中有哪些方案?**
A:Flex、Grid、absolute+transform(translate 百分比基于自身)。负 margin 法需要已知尺寸,不适用。

## 一句话背诵版

**首选 Flex(justify-content + align-items: center)或 Grid(place-items: center);不定宽高兼容方案用 absolute + translate(-50%,-50%);定宽高可用 absolute + margin auto 或负 margin。**

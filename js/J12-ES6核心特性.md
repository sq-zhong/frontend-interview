# J12 · ES6 核心特性

> 热度:🔥🔥

## 一句话考点

ES6(ES2015)是 JS 的里程碑版本,面试高频点:`let/const`、箭头函数、解构、模板字符串、`Promise`、`class`、模块化、`Set/Map`、扩展运算符、`Symbol`。

## 原理精讲

### 1. let / const(见 J3)

块级作用域、无变量提升(TDZ)、不可重复声明;const 声明后引用不可变。

### 2. 解构赋值

```js
const [a, b, ...rest] = [1, 2, 3, 4];   // a=1 b=2 rest=[3,4]
const { name, age = 18 } = user;         // 默认值
const { data: list } = res;              // 重命名
function f({ x, y }) {}                   // 参数解构
```

### 3. 扩展 / rest 运算符

```js
const merged = { ...obj1, ...obj2 };     // 对象合并
const copy = [...arr];                    // 数组浅拷贝
const max = Math.max(...nums);            // 展开传参
function sum(...args) {}                   // 收集参数为数组
```

### 4. Set / Map

```js
// Set:值唯一,常用于去重
const unique = [...new Set([1, 1, 2, 3])];  // [1,2,3]

// Map:任意类型作键,有序,可遍历
const m = new Map();
m.set('key', 1).set(obj, 2);
m.size; m.has('key'); m.get(obj);
```

### 5. Symbol

独一无二的原始值,用作对象唯一属性键,避免命名冲突。

### 6. 其他常用

- 模板字符串:`` `${var}` ``,支持多行。
- 默认参数:`function f(a = 1) {}`。
- 对象简写:`{ name, getName() {} }`。
- `for...of`:遍历可迭代对象的值(区别于 `for...in` 遍历键)。
- 可选链 `?.`(ES2020)、空值合并 `??`(ES2020)。

## 代码示例

```js
// 可选链 + 空值合并(高频实用)
const city = user?.address?.city ?? '未知';
// user 或 address 为 null/undefined 时不报错,返回 '未知'
// ?? 仅在 null/undefined 时取默认(区别于 ||,0 和 '' 不会走默认)

0 || 'default';    // 'default'(0 被当假值)
0 ?? 'default';    // 0(只有 null/undefined 才走默认)

// for...of vs for...in
const arr = ['a', 'b'];
for (const v of arr) console.log(v);   // 'a' 'b'(值)
for (const k in arr) console.log(k);   // '0' '1'(键/索引)

// 数组去重与合并
const dedup = [...new Set(arr)];
const flat = [1, [2, [3]]].flat(Infinity);   // [1,2,3]
```

## 高频追问

**Q:`??` 和 `||` 区别?**
A:`||` 在左侧为任意假值(0、''、false、null、undefined、NaN)时走右侧;`??` 只在 null/undefined 时走右侧,能保留 0 和空字符串等有效值。

**Q:`for...of` 和 `for...in` 区别?**
A:for...in 遍历可枚举的**键**(含原型链上的,适合对象但顺序不保证);for...of 遍历可迭代对象的**值**(数组、Set、Map、字符串),不能直接遍历普通对象。

**Q:Set 如何去重,有什么局限?**
A:`[...new Set(arr)]`。局限:基于 `SameValueZero` 比较,对象/数组等引用类型无法按内容去重(引用不同即视为不同)。

**Q:ES6 模块和 CommonJS 区别?**
A:ES6 `import/export` 是**静态**(编译时确定依赖,可 tree-shaking)、值是**引用**(动态绑定);CommonJS `require` 是**运行时**加载、值是**拷贝**、同步加载。

## 一句话背诵版

**ES6 高频:let/const 块级作用域、箭头函数、解构、扩展运算符、模板字符串、Promise、class、模块化、Set 去重/Map 任意键;`??` 只对 null/undefined 生效(保留 0/''),`?.` 安全取值;for...of 遍历值、for...in 遍历键。**
